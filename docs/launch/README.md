# Launch kit

Ready-to-post copy and submission steps for getting MCP Widget Studio in front
of people. Personalize the bracketed bits before posting.

- `awesome-mcp.md` — entry text + how to submit the PR
- `show-hn.md` — Hacker News "Show HN" post
- `reddit.md` — r/mcp, r/ClaudeAI, r/LocalLLaMA variants
- `devto.md` — long-form blog post
- `x-thread.md` — X/Twitter launch thread

**Sequencing that works well:** dev.to post first (gives you a link to
reference) → awesome-mcp PR → Reddit → Show HN (save this for a day you can
reply to comments for a few hours) → X thread pointing at the HN post.
